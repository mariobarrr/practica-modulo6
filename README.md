# 🌐 Portfolio & Blog Personal

Proyecto desarrollado como práctica del Módulo 6.
Aplicación web construida con React y Node.js.

## 🛠️ Tecnologías

- React 18
- Node.js
- CSS Modules

## 🚀 Instalación
```bash
git clone https://github.com/TU_USUARIO/practica-modulo6.git
cd practica-modulo6
npm install
npm run dev
```

## 📖 Uso

Visita `http://localhost:3000` en tu navegador.
